# Netflix Landing Page Clone

A simple **Netflix Landing Page Clone** built using **HTML5** and **CSS3**. This project recreates the look and feel of the official Netflix landing page while focusing on responsive design and modern web layout techniques.

## 📌 Project Overview

This project is a frontend-only clone of the Netflix homepage. It is designed to practice HTML and CSS fundamentals, including layout creation, flexbox, responsive design, background images, and styling.

## 🚀 Features

- Responsive landing page design
- Modern Netflix-inspired UI
- Hero section with call-to-action
- Navigation bar
- Feature sections
- FAQ section layout
- Footer with links
- Clean and organized code structure

## 🛠️ Built With

- HTML5
- CSS3
